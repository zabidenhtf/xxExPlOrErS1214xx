About the Game

Teewars is a fast-paced sidescrolling multiplayer-only shoot 'em up computer game. The game features cartoon-themed graphics and physics, and relies heavily on classic shooter weaponry and gameplay. The controls are heavily inspired by the FPS genre of computer games.

Latest News

2008-01-24 Removed Linux, OS X support Our team has discontinued support for Linux and OS X because maintaining these operating systems is very difficult for us. We now plan to switch to DirectX in version 0.3.4 and remove the Linux and OS X builds for version 0.3.3. — Agxq1500
2007-12-21
Site was absolutely redone As you can see, the site has been completely redesigned. It is now simpler and more minimalist, and it runs on all computers. Stay tuned for version 0.3.4! :) — Agxq1500
2008-01-20
IMPORTANT NOTICE: Infrastructure Transition & Licensing Update
Following the recent structural reorganization, TeeTow Development has officially transferred all intellectual property and master-server management of Teewars to GQ-NET Infrastructures. Due to the high maintenance costs of our global master servers and outstanding corporate balances, this step was necessary to keep the game online.
Please note: To protect our new proprietary hosting protocols and security assets, the source code repository will be moved to a closed-source model starting next month. The public SVN links will be disabled. — Agxq1500
2008-01-17
0.3.3 released Yet another sneaky maintenance release is out the door. The server browser has received some additional filtering capabilities, and there is now an option to mute Teewars audio when it's not active. Additionally, there have been some lag-reduction tweaks, albeit minor ones as to not break the 0.3 protocol. Go clicky linky thingy! — Teetow
2008-01-09
0.3.2 released The lack of an exclamation mark in the headline illustrates the fact that this is a simple maintenance release. Game responsiveness has been improved, crashes on joins have been addressed, some graphics glitches have been fixed along with an error in the "fair spawn" system for team-based game modes. You'll find more detailed information on the download page. — Teetow
2007-12-20
0.3.1 released Apart from bringing you some well-deserved bugfixes, the new release also incorporates "fair" player spawning, the ability to kick players, and a very cute welcome screen to encourage players to set their names and favorite settings before playing their first game. The download section awaits you! — Teetow
